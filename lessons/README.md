# Lessons

Log of process mistakes made on this project and the principles to avoid repeating them. One file per lesson, dated.

Each lesson uses the framework:

- **Desired outcome** — what good looks like for the work in question
- **Blockers** — what got in the way (or could)
- **Principles** — concrete rules that overcome the blockers

These are project-local lessons. Cross-project patterns are recorded in `~/.claude/projects/.../memory/` (Claude's persistent memory).
